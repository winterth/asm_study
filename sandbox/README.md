###GDB

###通用寄存器
eax, ebx, ecx, edx
```
                 EAX
+--------+--------+--------+--------+
|        |        |        |        |
+--------+--------+--------+--------+
^                 ^   AH   ^   AL   ^
                          AX
```
#####复值指令：
######mov
```
mov eax, 1234H
```
dst和src的长度必需一致
```
mov eax, bx   <-- error
mov eax, ebx  <-- OK
```

######add
```
add ax, 10H
```
相当于 `ax=ax+10H`

######sub
```
sub ax, 10H
```
相当于 `ax=ax-10H`

###执行下一条指令
```
CS:IP
```
CS的值左移4位 + IP的值 ＝ 下一条指令在内存中的地址
修改下一条指令的地址有2种方法：

1. 自动。执行完一条后，IP的值会自动加上当前指令的长度
2. 用`jmp`跳转

	```
	jmp 100H       <-- 在当前段中跳转到100H的地方
	jmp 2000H:10H  <-- 直接跳转到指定的地址，即200010H
	```
